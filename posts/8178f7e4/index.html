一、前言

在4k广告机中需要实现多语言切换，这时候接触到国际化，前端框架无数，其中几种热门的框架都有相匹配的国际化插件工具。比如：

  • vue + vue-i18n
  • angular + angular-translate
  • react + react-intl

  • jquery + jquery.i18n.property

    在4K广告机项目中使用的前端框架为Vue，故而我们将使用vue-i18n这个插件进行国际化功能的实现。
